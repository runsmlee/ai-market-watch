## 🚀 실행 준비 상태

### 필요한 단계
1. **Node.js 설치** (v18 이상)
2. **의존성 설치**: `npm install`
3. **환경 변수**: `.env.local` 파일 생성
4. **개발 서버**: `npm run dev`
